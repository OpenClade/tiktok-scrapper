# Tiktok scrappper module

## Installation

- Create virtual environment
```python3 -m venv venv```


- Activate virtual environment
```source venv/bin/activate```

- Install requirements
```pip install -r requirements.txt```

## Usage
``` python3 tiktok_scrapper.py ```
